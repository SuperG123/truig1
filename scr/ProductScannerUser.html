TRU Industrial Group

Project: Product Scanner

Subject: Product Scanner Documentation and Use

Target: Application Users

User Log-In

On the “home” page for the product scanner the user will be presented with the option to log-in and use the product scanner, read the help documentation or send an internal message.

1) Log-In

        The user will have been assigned a username and a password prior to accessing the product scanner application. The credentials assigned should be used to proceed through this section of the product scanner application. If the user has forgotten their log-in information they should contact their supervisor to have new credentials issued.

2) Help Documentation

        If the user has any questions regarding the use of the application their first course of action should be to review the “Help” section of the application. If a satisfactory answer is not found in the Help documentation the user should contact their supervisor for a solution to the issue.

3) Internal Messaging

        The product scanner application is equipped with an internal messaging system that will allow users to leave passive messages for other users of the application. If questions arise or if the user would like to leave a message for any reason the user may complete the message form and submit the message. This should not be used for messages of any urgency or emergency.

Scanning Products

Once a user has logged-in they are presented with a page allowing them to scan a product. The user may either enter a product UPC (bar code) or scan a product with a bar code scanner. If the product is found in the database the user is presented with the saved information for the product they scanned. If the product is not found in the database the user is presented with the opportunity to add the new product to the system.

1) Scan or type the UPC

        With your scanner scan the product bar code or type the product UPC code into the UPC box.

2) Product Action

        If the product is found enter the current quantity that is available at your location, and the location code for the product. If the product is not found complete the form for adding a new product to the database.

3) Save.

        When you click the save button the cursor should move back to the UPC entry box indicating the system is now ready for you to scan another product.

View Messages

After logging-in to the application a user is given the option to View Messages.

When the view message option is clicked the user is directed to the messages screen where all messages are listed allowing the user to easily view who sent the message, the message subject, a short portion of the message body, a button to read the complete message and a button to delete the message.

*note: there is not a backup or archive of messages. Once a message is deleted the message is, in fact, gone forever.
